\# 🔥 ESP8266 PC Hardware Monitor



A real-time PC hardware monitoring system built using an ESP8266 NodeMCU and SSD1306 OLED Display.



This project displays:



\- CPU Usage

\- CPU Temperature

\- GPU Usage

\- GPU Temperature

\- RAM Usage



directly on a 0.96/0.98 inch OLED screen using Python Serial Communication.

\# 🛠 Hardware Used



\- ESP8266 NodeMCU

\- SSD1306 OLED Display (I2C)

\- Jumper Wires

\- USB Cable

\# 🔌 OLED Connections



| OLED | NodeMCU |

|------|------|

| VCC | 3.3V |

| GND | GND |

| SCL | D1 |

| SDA | D2 |

\# ⚡ How It Works



1\. Python reads hardware sensor data using OpenHardwareMonitor.

2\. Data is sent to ESP8266 through Serial communication.

3\. ESP8266 updates the OLED display in real time.
